Useful Grid classes

DXP Cloud components like management-toolbar, breadcrumbs or toolbar-labels have a main 32px horiontal padding, to add this behaviour in grids you can add:

.container-padded

If in otherwise what you want is removing the horiontal padding you can use:

.container-glued

If you want is to remove the horiontal padding only in XS devices:

.container-xs-glued

Lists

Tabular List

Align content inside a list-group like a table with .tabular-list-group.

.list-group-item behaves as a table row.

Use a combination of .list-group-item-field and .list-group-item-content to control the size of each cell. .list-group-item-field has width: 1% and will expand only as wide as its content as long as .list-group-item-content is present as a sibling.

.list-group-item-content will expand to fill the remaining space.

Table List

A table styled like a list. The active state can be invoked by adding class active to the <tr></tr> element. Alse exist diferents states to add to the table like table-actions if you want to add the last column with actionstable-autobix to set the table auto fitable, table-checkbox when we want to use a checkbox at the first column table-out-border to add a single border, table-stripe to add striped lines, table-heading-nowrap to add no wrap effect and table-hover

Actions column can be added using the classactions in the last <tr></tr> of table body
