Table List

A table styled like a list. The active state can be invoked by adding class active to the `````` element. Alse exist diferents states to add to the table like table-actions if you want to add the last column with actionstable-autobix to set the table auto fitable, table-checkbox when we want to use a checkbox at the first column table-out-border to add a single border, table-stripe to add striped lines, table-heading-nowrap to add no wrap effect and table-hover

Actions column can be added using the classactions in the last `````` of table body

Fixed column and table head, table-fixed class has to be added in table-responsive div.

To fix the head we can create a copy of the table which contains only the thead, this table-copy will have to contain table-fixed-head class

To fix columns we should create a copy of the table which contains one or two columns, this table-copy will have to contain table-fixed-column class. Columns fixed will have a width limit: 240px in desktop, 33% in mobile.
